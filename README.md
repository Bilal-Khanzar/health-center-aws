# Health Center - AWS Deployment Project

This project demonstrates the deployment of a professional health center web portal on an **AWS EC2** instance using an **Apache2 (HTTPD)** web server.

## 🚀 Key Features
* Hosted on **AWS Cloud** (EC2 Instance).
* Managed via **Linux Command Line (CLI)**.
* Secured with **SSH Key-based authentication** for Git operations.
* Deployed using **Apache HTTP Server**.

## 🛠️ Tech Stack
* **Cloud:** AWS (EC2, Security Groups)
* **OS:** Linux (Ubuntu/Amazon Linux)
* **Web Server:** Apache2
* **Version Control:** Git & GitHub
